INTRODUCTION:
The Ambulance Tracking System is a Java-based application designed to help users quickly find the nearest available ambulance during emergencies. The system stores ambulance details such as location and availability status, and uses a simple distance calculation method to identify the closest ambulance to the user’s location.
FEATURES:
🚑 Stores multiple ambulance details
📍 Uses latitude and longitude for location tracking
📊 Calculates distance using a mathematical formula
✅ Filters only AVAILABLE ambulances
🔍 Finds the nearest ambulance efficiently
⚡ Simple and fast execution
🧩 Easy to understand and modify
How IT PROCEED:
The program uses Object-Oriented Programming (OOP) concepts:
Class & Object → Ambulance, Location
Encapsulation → Data stored inside objects
Uses ArrayList from Java Collections to store ambulances
Distance is calculated using Euclidean formula:
